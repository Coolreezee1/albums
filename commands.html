Moderation 18 commands

/warn
Issue a tracked warning to a member and log it under a case number.
/warnings
View a member's full warning history.
/mute
Timeout a member for {duration}, with an optional reason.
/unmute
Remove an active timeout early.
/kick
Remove a member from the server.
/ban
Ban a member and optionally delete their recent messages.
/unban
Lift a ban using a user ID.
/purge
Bulk-delete up to {amount} messages from a channel.
/automod enable
Turn on a protection module: raid detection, link filter, or spam filter.
/automod disable
Turn off a protection module.
/modlogs
Set the channel where moderation actions are recorded.
/verify-panel
Send a panel where new members verify themselves.
/verify-role
Set the role granted once a member verifies.
/lockdown
Lock a channel so only staff roles can send messages.
/slowmode
Set a per-member cooldown between messages in a channel.
/case view
Look up the details of a specific moderation case.
/case delete
Remove a case from a member's record.
/blacklist word
Add a word or phrase to the auto-filtered list.
