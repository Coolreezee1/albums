Capabilities

Everything your server needs, wrapped up cute

Four little departments, one bot. Set it up once and Kaede takes it from there.

Moderation

Warnings, mutes, and raid detection that log everything and never lose a case number.

/warn

Issue a tracked warning and log it to the case history.

/automod enable

Turn on a protection module like raid or spam detection.

/modlogs

Set the channel where moderation actions are recorded.

Automation

Reaction roles, scheduled announcements, and triggers that fire while you're offline.

/reaction-role add

Bind a role to an emoji on a message.

/schedule

Queue a message to post in a channel at a set time.

/trigger add

Auto-respond when a keyword appears in chat.

Leveling

Give members a reason to stick around, with ranks that track real activity.

/level

Show a member's current rank and XP progress.

/leaderboard

Display the server's top-ranked members.

/level-role set

Grant a role automatically at a chosen level.

Utility

The everyday commands members actually reach for.

/poll

Post a reaction-based poll with up to 10 options.

/remind

Set a personal reminder delivered by DM.

/serverinfo

Show member count, boost tier, and creation date.
